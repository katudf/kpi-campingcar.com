貸出当日の流れFlow

※当日までに必ず、使用方法、返却時の注意事項、
キャンピングカーの使用マナー等を、ご一読お願いします。

  1. 貸出時間の30分前を目安にお越しください。

  2. 貸渡証等の書類に記入をお願いします。

    ※料金は現金前払い制となっております。
    スムーズに進められるよう、お釣りがないようにご協力お願いいたします。

  3. 出発前に車両の説明を致します。

    ※不安な点やご質問等ございましたらお気軽にお尋ねください。

  4. お荷物を積み込み、車のご移動

  5. 出発

貸出時の注意事項Lending Rules

  • 車内飲食は可能ですが、ニオイの残るような食べ物の飲食は厳禁です。
    （特に焼肉等）

  • 車内調理は不可になります。
    キッチンはお湯を沸かす程度のご使用でお願いします。

  • 運転歴5年以上としております。
    (ペーパードライバーの方はご遠慮いただくようお願いします。)

  • 直寝は禁止です。
    衛生面を考慮して、寝袋、シーツ、布団等を必ずご持参ください。

  • 燃料(軽油)満タンでの返却となっております。

返却時の流れFlow of return

貸し出した時と同じ場所に駐車してください

 

  • 平日の場合

  • 担当者と一緒に外装・内装のチェック

  • チェック表の記入

  • 備品の返却チェック。チェック表の記入

  • お忘れ物のご確認

  • 返却終了

  • 休日の場合

  • セルフチェック表に基づき、
    外装・内装のセルフチェックをお願いします。
    セルフチェック表の記入をお願いします。

  • 貸出備品のチェック。
    チェック表の記入をお願いします。

  • メインスイッチが切ってある事、
    鍵が閉まっている事を確認し、
    セルフチェック表、備品チェック表と
    車両の鍵をポストへ返却

  • 備品は車両に乗せたままで構いません。

  • 返却終了

返却時の注意事項Return Rules

  • 車体の傷・車内の汚れをご確認ください。
    (チェックシートに記入をお願い致します)

  • 燃料(軽油)満タン返しでお願い致します。

  • 忘れ物がないかご確認下さい。

  • トイレ使用した場合、お支払いや処理が済んでいるかご確認ください。
